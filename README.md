# FUTURE_PE_02
Smart Water Bottle

Problem Addressed
Students, professionals, and fitness enthusiasts often forget to drink enough water throughout the day, leading to dehydration, fatigue, headaches, and reduced productivity.
Prompt Framework


Prompt 1 — Hook Creation
Prompt:
Generate 10 short, attention-grabbing UGC hooks for a Smart Water Bottle. The hooks should focus on hydration problems, productivity, health benefits, and personal experiences. Keep them natural, relatable, and suitable for Instagram Reels, TikTok, and YouTube Shorts.


Prompt 2 — Script Creation
Prompt:
Create a 30-second UGC-style advertisement for a Smart Water Bottle using one of the generated hooks. 

Follow the structure:
Hook
Problem
Solution

Personal Benefit
Call-to-Action
Use a friendly and authentic tone that feels like a genuine recommendation rather than a sales pitch.

Produces authentic UGC content that feels personal and trustworthy.

Uses a clear Problem → Solution → Benefit → CTA structure.
Works across Instagram, TikTok, and YouTube Shorts.
Easily adaptable to other products such as fitness trackers, ergonomic chairs, meal planners, skincare products, or language-learning apps.
